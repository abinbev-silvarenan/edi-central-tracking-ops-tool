# EDI Central Tracking — Operations (prototype)

Static **hypothesis prototype** for the vendor-operations EDI central tracking operational tool (Hexa / BEES-style shell). See `HYPOTHESIS.md` for scope and metrics.

## Live site (GitHub Pages)

After enabling Pages (**Settings → Pages → Branch `main` / `/ (root)`**), the app is available at:

**https://silarerenan-cmyk.github.io/edi-central-tracking-ops-tool/**

This repository includes a `.nojekyll` file so the `_shared/` assets folder is served correctly (Jekyll would otherwise skip paths starting with `_`).

## Local preview

Open `index.html` in a browser, or serve this directory with any static file server.

## Analytics dashboard

Open `_shared/dashboard.html?id=edi-central-tracking-ops-tool` (same origin as the prototype) to view click tracking for this build.
